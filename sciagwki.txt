git branch - lista gałęzi
git fetch --all pobranie zdalnych gałęzi
git branch --all wyświetlenie lokalnych i zdalnych gałęzi
git branch <nazwa gałęzi> - stworzenie nowej
git branch -d <nazwa gałęzi> - usunięcie gałęzi
git checkout <nazwa gałęzi> - zmiana gałęzi
git checkout -b <nazwa gałęzi> - stworzenie nowej i zmiana gałęzi
git push origin <nazwa gałęzi> - pushowanie nie do mastera
git push -u origin <nazwa gałęzi> - ustawienie gałęzi jako głównej w celu używania komendy git push

Przechodzimy na develop
git merge <nazwa gałęzi, którą chcemy scalić>
